</p>

## Getting started

### Installation

```shell
git clone git@github.com:jakubwiczkowski/its-a-match.git
cd its-a-match
npm ci
```

## Usage

Run in development mode:

```bash
npm run dev
```

Run in production mode:

```bash
npm run build 
npm run start
```
